# 📞 Contact Book in C

A simple file-based contact book project written in C.

## 🔧 Features
- Add Contact
- View Contacts
- Search Contact
- Delete Contact
- File-based storage (persistent data)

## 💻 Technologies Used
- C Programming
- File Handling (fprintf, fscanf)
- Structures

## ▶️ How to Run

```bash
gcc contact.c -o ContactBook.exe
ContactBook.exe
